# 遊戲中心 - 貪吃蛇與方塊數獨

## 項目概述

這是一個遊戲中心主頁，整合了兩個經典遊戲和兩種載入畫面主題。主頁作為遊戲的統一入口，提供現代化的用戶體驗。

## 功能特色

### 1. 雙主題設計
- **深色模式**：科技感十足的深色主題，適合夜間遊戲體驗
- **明亮模式**：清新明亮的白色主題，適合日間遊戲體驗
- **自動保存**：使用localStorage保存用戶主題偏好
- **系統適配**：自動檢測操作系統主題偏好

### 2. 載入畫面預覽
- **互動預覽**：兩種載入畫面的靜態/動態預覽
- **一鍵切換**：可在靜態預覽和iframe動態預覽之間切換
- **完整查看**：可在新標籤頁中查看完整的載入畫面

### 3. 遊戲選擇
- **貪吃蛇遊戲**：經典的貪吃蛇遊戲，包含多種控制方式和速度調整
- **方塊數獨**：益智消除遊戲，挑戰邏輯思維
- **遊戲說明**：每個遊戲都有詳細的規則和說明

### 4. 響應式設計
- **桌面版**：完整功能與控制
- **平板版**：優化佈局與觸控控制
- **手機版**：簡化界面與虛擬方向鍵

## 技術實現

### 前端技術
- **HTML5**：頁面結構
- **CSS3**：樣式設計與響應式佈局
- **JavaScript**：互動功能與主題切換
- **Tailwind CSS**：現代化CSS框架
- **Google Fonts**：字體資源
- **Material Icons**：圖標資源

### 主要功能實現
1. **主題切換系統**
   - 使用CSS變量和Tailwind的dark模式
   - localStorage保存用戶偏好
   - 平滑的過渡動畫

2. **載入畫面預覽系統**
   - iframe嵌入技術
   - 靜態/動態預覽切換
   - 響應式設計適配

3. **遊戲導航系統**
   - 平滑滾動導航
   - 遊戲說明模態框
   - 直接遊戲啟動連結

## 使用說明

### 快速開始
1. 直接打開 `index.html` 文件
2. 選擇您想玩的遊戲
3. 點擊"開始遊戲"按鈕

### 主題切換
- 點擊右上角的主題切換按鈕
- 系統會自動保存您的偏好
- 下次訪問時會自動應用您的主題

### 載入畫面預覽
- 點擊"切換預覽模式"按鈕切換靜態/動態預覽
- 點擊"查看完整畫面"在新標籤頁中查看完整載入畫面

### 遊戲說明
- 點擊遊戲卡片上的"遊戲說明"按鈕
- 查看詳細的遊戲規則和操作說明
- 點擊"關閉"按鈕關閉說明窗口
