# [wengcheng]'s Game Center - 產品需求規格書 (PRD)

## 1. 專案概述

### 1.1 專案名稱
[wengcheng]'s Game Center (遊戲中心)

### 1.2 專案目標
建立一個簡約現代、色彩明艷的遊戲中心網站，提供多款休閒遊戲，讓用戶在輕鬆愉快的環境中享受遊戲樂趣。

### 1.3 目標用戶
- 喜歡休閒遊戲的用戶
- 尋求放鬆娛樂的用戶
- 各年齡層的遊戲愛好者

## 2. 設計規範

### 2.1 設計風格
- **整體風格**: 簡約現代，色彩明艷
- **視覺感受**: 令人輕鬆、愉快
- **語言**: 繁體中文

### 2.2 色彩方案
- **主色調**: 
  - 蛇身: 霓虹色或綠色，可使用多色漸層增加視覺刺激
  - 食物: 白色或亮色
  - 背景: 淺色或深色（突出蛇身）或豐富的背景圖案，或灰色作背景

### 2.3 主題模式
- **黑暗模式**: 深色背景，亮色文字和元素
- **明亮模式**: 淺色背景，深色文字和元素
- **切換功能**: 用戶可隨時切換兩種模式

## 3. 頁面結構

### 3.1 Header (頁首)
- **網站標題**: 貪吃蛇lets go
- **標語**: 貪吃蛇喜歡吃！ (簡短 slogan)

### 3.2 遊戲卡片區
每張卡片包含以下元素：
1. **遊戲圖示/預覽**: 貪吃蛇圖示或遊戲截圖
2. **遊戲名稱**: 遊戲標題
3. **簡介**: 遊戲簡短描述
4. **開始按鈕**: 啟動遊戲的按鈕

### 3.3 Footer (頁尾)
- **版權資訊**: © 2026 [wengcheng]'s Game Center
- **創作者**: wengcheng

## 4. 遊戲列表

### 4.1 現有遊戲

#### 遊戲1: 貪吃蛇遊戲
- **描述**: 經典的貪吃蛇遊戲，控制蛇吃食物成長，避免撞牆或撞到自己身體
- **檔案**: `snake.html`
- **特色**:
  - 可調整遊戲速度
  - 移動設備觸控控制
  - 本地儲存最高分數
  - 響應式設計：支援手機、桌面、平板

#### 遊戲2: 方塊數獨
- **描述**: 拖曳方塊到棋盤上，填滿整行或整列即可消除得分
- **檔案**: `block blash.html`
- **特色**:
  - 多種方塊形狀
  - 連擊得分系統
  - 提示功能
  - 旋轉方塊功能
  - 完整響應式設計

### 4.2 未來可擴充遊戲

## 5. 功能需求

### 5.1 核心功能
1. **遊戲中心首頁**: 展示所有遊戲卡片
2. **遊戲載入**: 點擊卡片直接載入遊戲
3. **主題切換**: 黑暗/明亮模式切換
4. **響應式設計**: 支援各種設備尺寸

### 5.2 用戶體驗需求
1. **快速載入**: 遊戲頁面快速載入
2. **直觀操作**: 簡單易懂的遊戲控制
3. **視覺回饋**: 清晰的遊戲狀態提示
4. **分數記錄**: 本地儲存遊戲記錄

## 6. 技術規格

### 6.1 前端技術
- **HTML5**: 遊戲頁面結構
- **CSS3**: 樣式設計，包括黑暗/明亮主題
- **JavaScript**: 遊戲邏輯和互動功能
- **Canvas**: 遊戲圖形渲染

### 6.2 瀏覽器相容性
- Chrome (最新版本)
- Firefox (最新版本)
- Safari (最新版本)
- Edge (最新版本)

### 6.3 設備相容性
- 桌面電腦 (≥ 1024px)
- 平板電腦 (768px - 1023px)
- 手機 (< 768px)

## 9. 成功指標

### 9.1 技術指標
- 頁面載入時間 < 3秒
- 遊戲切換時間 < 2秒
- 100% 響應式設計

### 9.2 用戶指標
- 用戶平均遊戲時間 > 5分鐘
- 主題切換使用率 > 30%
- 用戶回訪率 > 40%

## 10. 詳細遊戲規格

### 10.1 遊戲專用PRD文件
每個遊戲都有獨立的詳細產品需求規格書：

1. **貪吃蛇遊戲詳細PRD**: `snake-game-PRD.md`
   - 包含完整的遊戲機制、技術規格、用戶界面設計
   - 詳細的遊戲規則與得分系統
   - 設備相容性與測試要求

2. **方塊數獨遊戲詳細PRD**: `block-blast-PRD.md`
   - 包含完整的益智遊戲設計
   - 方塊系統與消除機制詳解
   - 等級系統與得分計算細節

### 10.2 注意事項
1. 所有遊戲需保持一致的設計語言
2. 主題切換需無縫過渡
3. 確保遊戲狀態在頁面切換時正確保存
