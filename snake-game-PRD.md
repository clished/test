# 貪吃蛇遊戲 - 產品需求規格書 (PRD)

## 1. 遊戲概述

### 1.1 遊戲名稱
貪吃蛇遊戲 (Snake Game)

### 1.2 遊戲類型
經典休閒遊戲

### 1.3 遊戲目標
控制蛇吃食物成長，獲得高分，避免撞牆或撞到自己身體。

### 1.4 目標玩家
- 所有年齡層的玩家
- 喜歡經典休閒遊戲的用戶
- 尋求短時間娛樂的玩家

## 2. 遊戲功能

### 2.1 核心遊戲機制
1. **蛇移動**: 蛇持續向前移動，玩家控制方向
2. **食物收集**: 蛇頭接觸食物時，蛇身增長，分數增加
3. **碰撞檢測**: 檢測與牆壁和自身身體的碰撞
4. **分數系統**: 每吃一個食物得10分

### 2.2 遊戲控制
- **鍵盤控制**: 方向鍵或WASD
- **移動設備控制**: 觸控方向按鈕
- **快捷鍵**:
  - 空格鍵: 開始/繼續遊戲
  - P鍵: 暫停/繼續遊戲
  - R鍵: 重新開始遊戲

### 2.3 遊戲狀態
- **準備狀態**: 遊戲開始前
- **進行狀態**: 遊戲進行中
- **暫停狀態**: 遊戲暫停
- **結束狀態**: 遊戲結束

## 3. 技術規格

### 3.1 開發技術
- **HTML5**: 遊戲頁面結構
- **CSS3**: 樣式設計與響應式佈局
- **JavaScript**: 遊戲邏輯與互動
- **Canvas API**: 遊戲圖形渲染

### 3.2 性能要求
- **幀率**: 60 FPS
- **載入時間**: < 2秒
- **響應時間**: 控制輸入延遲 < 100ms

### 3.3 儲存功能
- **本地儲存**: 使用localStorage儲存最高分數
- **遊戲狀態**: 暫停時保存當前遊戲狀態

## 4. 用戶界面設計

### 4.1 遊戲畫面元素
1. **遊戲畫布**: 500x500像素遊戲區域
2. **分數顯示**: 當前分數與最高分數
3. **控制按鈕**: 開始、暫停、重新開始按鈕
4. **速度控制**: 遊戲速度調整滑桿
5. **狀態提示**: 遊戲狀態訊息顯示

### 4.2 視覺設計
- **蛇身**: 霓虹綠色漸層，頭部有眼睛
- **食物**: 紅色發光圓形
- **背景**: 深色漸層背景
- **網格**: 半透明網格線輔助定位

### 4.3 響應式設計
- **桌面版**: 完整功能與控制
- **平板版**: 優化佈局與觸控控制
- **手機版**: 簡化界面與虛擬方向鍵

## 5. 遊戲規則

### 5.1 基本規則
1. 蛇不能停止移動
2. 蛇不能直接反向移動（例如從右轉左）
3. 吃到食物後蛇身增長一格
4. 撞牆或撞到自己身體遊戲結束

### 5.2 得分規則
- 每個食物: +10分
- 每50分自動增加遊戲速度
- 打破最高分記錄時有特殊提示

### 5.3 難度設定
- **速度調整**: 玩家可手動調整遊戲速度
- **自動加速**: 每得50分自動加速
- **難度等級**: 可考慮未來添加難度選擇

## 6. 音效與動畫

### 6.1 音效需求
- **吃食物音效**: 簡短清脆的音效
- **碰撞音效**: 遊戲結束提示音
- **按鈕音效**: 按鈕點擊反饋

### 6.2 視覺效果
- **食物發光效果**: 脈衝發光動畫
- **蛇移動效果**: 平滑移動過渡
- **分數彈出**: 得分時數字彈出效果
- **遊戲結束效果**: 閃爍或震動效果
